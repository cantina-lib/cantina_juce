
## Cantina plugin

### Info

Cantina as an audio plug-in.
Working only on LV2 for now.

### Build 

#### Instructions

Using cmake (>= 3.12), from the project's root directory:
  
    mkdir build
    cd build
    cmake ..
    make

#### Dependencies 

* Cantina (submodule)
* JUCE
* LV2
* Sord (LV2 validation)
